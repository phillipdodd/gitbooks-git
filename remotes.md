# Remotes

Official Documentation [here](https://git-scm.com/book/en/v2/Git-Basics-Working-with-Remotes).

## Show Remotes

`git remove -v`

## Add Remote \(to existing Repo\)

`git remote <remote url>`

{% hint style="success" %}
This uses the **same url**  that would be used for `git clone`
{% endhint %}
